# MyCommandTool
A simple command-line tool based on C++

## 1.使用说明
* 下载MyCommandTool.msi进行安装，该文件由Visual Studio 2022打包生成
* 阅读 **MyCommandTool_使用说明&结果展示** 并利用已有.cvs文件进行命令行操作
